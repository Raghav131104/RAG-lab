Introduction
Breadth First Search (BFS) is a graph traversal algorithm that explores vertices in the order of their distance from the source vertex. It starts at a given vertex and explores all vertices at the present depth level before moving on to vertices at the next depth level. BFS uses a queue data structure to keep track of the vertices that need to be explored.
Key Characteristics
• Uses FIFO (First In First Out) principle through a queue
• Explores vertices level by level
• Guarantees shortest path in unweighted graphs
• Requires more memory compared to DFS due to queue storage
• Complete algorithm - will find solution if it exists
Algorithm Steps
1. Start with a source vertex and mark it as visited
2. Add the source vertex to the queue
3. While the queue is not empty:
a. Dequeue a vertex from the front of the queue
b. Process the dequeued vertex
c. For all adjacent vertices of the dequeued vertex:
i. If the adjacent vertex is not visited, mark it as visited and enqueue it
4. Repeat step 3 until the queue becomes empty
Time and Space Complexity
Time Complexity: O(V + E) where V is the number of vertices and E is the number of edges
Space Complexity: O(V) for the queue and visited array
