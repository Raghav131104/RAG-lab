LangGraph is a framework for building stateful, multi-agent LLM applications.
FAISS is a vector database for efficient similarity search.
RAG reduces hallucinations by grounding LLMs in retrieved documents.
Transformers generate embeddings for semantic search.
LangChain orchestrates LLM tools and agents.

PROJECT IDENTITY:
This Multi-Agent RAG System was developed by Raghav Kumar. 
It uses a LangGraph-based state machine to coordinate agents.

CORE ARCHITECTURE DATA:
- Orchestrator: LangGraph State Machine.
- Retrieval Database: FAISS (Facebook AI Similarity Search) using local CPU indexing.
- Embedding Model: sentence-transformers/all-MiniLM-L6-v2.
- Generation Model: OpenAI GPT-4o-mini.
- Optimization: The system uses a "RecursiveCharacterTextSplitter" with 500-character chunks and 50-character overlap to maximize semantic context.

SYSTEM PERFORMANCE METRICS:
- Average Retrieval Accuracy: 92% based on internal benchmarking of technical documentation.
- Hallucination Reduction: 40% improvement over standard LLM calls by using "Admit Ignorance" prompting logic.
- Latency: Sub-second response time (typically <800ms) for end-to-end processing.

SECURITY & DEPLOYMENT:
- The system is built for local execution but is container-ready for Docker deployment.
- It leverages an "is_offline_mode" utility for secure, localized vector searching.
